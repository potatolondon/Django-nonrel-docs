Django nonrelational
=====================

These are the unofficial docs for Django nonrel. Django nonrel is
developed by `allbuttonspressed`_.

The `source for these docs`_ is available on Github, patches are welcome.

.. _allbuttonspressed: http://www.allbuttonspressed.com/
.. _source for these docs: https://github.com/potatolondon/Django-nonrel-docs

Getting help
============

* Try putting a message on the `Django non-relational mailing list`_. 

* See if there's a ticket for it in the `bug tracker`_.

* Ask about it in the `#django-nonrel IRC channel`_ on Freenode.

.. _Django non-relational mailing list: http://groups.google.com/group/django-non-relational/
.. _bug tracker: https://bitbucket.org/wkornewald/djangoappengine/issues
.. _#django-nonrel IRC channel: irc://irc.freenode.net/django-nonrel
